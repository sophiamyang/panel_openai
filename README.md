## How to run?

1. Get your OpenAI API key from openai.com and add your keys to the script. 

2. Install needed packages: \
`pip install openai`\
`conda install panel`

3. Run Panel dashboard:\
`panel serve panel_openai.ipynb` \
or \
`panel serve panel_openai.py`
